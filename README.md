## 📖 简介

本仓库是我在学习 C 语言过程中整理的个人笔记合集，以 **Markdown（.md）** 格式编写，便于阅读、检索和版本管理。

笔记涵盖 C 语言的基础语法、核心概念及常见编程技巧，适合：

- 🔰 C 语言初学者参考学习
- 🔄 复习巩固知识点
- 📂 作为个人知识库长期维护

## 🚀 如何使用

### 在线阅读
直接在 GitHub 上点击对应的 `.md` 文件即可在线阅读。

### 本地阅读
```bash
git clone https://github.com/Day-Dreamer2005/C-note-md.git
```
克隆到本地后，使用任意 Markdown 编辑器（如 VS Code、Typora、Obsidian）打开阅读。
