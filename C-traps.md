### 关于`strlen()`函数的返回值  
库函数`strlen()`的返回值为`size_t`类型（即`unsigned int`），其返回类型计算时结果仍为`size_t`类型  

如:
```C
int main()
{
	if (strlen("abc") - strlen("abcd") > 0)
	{
		printf("hehe\n");
	}
	else
	{
		printf("haha\n");
	}

	return 0;
}
```
程序最终输出结果为**hehe**✅

### ⚠️关于`=`与`==`的陷阱❗❗❗
`=`为运算操作符，意为将右值赋给左边（如 `a = 0`），`==`为判断操作符，意为判断两侧值是否相等（如 `a == 0`）
在例如循环的条件判断中，经常会用到`==`来判断条件是否成立，为误写为`=`,可以将常量放到`==`右侧（如 `0 == a`）来判断，因为`0 = a`是错误语法（变量不可能赋给常量），会报错，使得程序提前发现错误且便于调试

### 在 C 语言中，`''` 和 `""` 是**完全不同的两种东西**

| 对比项 | `'A'` | `"A"`      |
| --- | ----- | ---------- |
| 类型  | char  | char*（字符串） |
| 内容  | 一个字符  | 字符串        |
| 长度  | 1     | 2（含`\0`）   |
| 内存  | 一个字节  | 多个字节       |
| 用途  | 存字符   | 存文本        |

🔥 进阶理解（非常重要）

`" "` 不只是“一个空格”，而是：
```C
" "  == {' ', '\0'}
```

### void类型指针注意事项⚠️⚠️⚠️

#### 1.不能直接解引用
```C
int a = 10;
void* p = &a;

// ❌ 错误：不能直接解引用 void*
// printf("%d\n", *p);

// ✅ 正确：必须先转换为具体类型
printf("%d\n", *(int*)p);
```

#### 2.不能进行指针算术运算
```C
int arr[] = {1, 2, 3, 4, 5};
void* p = arr;

// ❌ 错误：不能对 void* 进行算术运算
// p++;  // 编译器不知道步长

// ✅ 正确：先转换类型
int* q = (int*)p;
q++;  // 步长为 sizeof(int)
```
**原因**：指针运算需要知道指向类型的大小，`void` 类型没有大小信息

#### 3.类型转换必须一致
```C
int a = 100;
void* p = &a;

// ✅ 正确：转换回原类型
int* pi = (int*)p;
printf("%d\n", *pi);  // 100

// ⚠️ 危险：转换错误的类型
double* pd = (double*)p;
printf("%f\n", *pd);  // 未定义行为！
```

#### 4.函数参数和返回值
`void*` 常用于实现泛型编程：
```C
// 泛型交换函数
void swap(void* a, void* b, size_t size) {
    char temp[size];
    memcpy(temp, a, size);
    memcpy(a, b, size);
    memcpy(b, temp, size);
}

// 使用
int x = 10, y = 20;
swap(&x, &y, sizeof(int));

double d1 = 3.14, d2 = 2.71;
swap(&d1, &d2, sizeof(double));
```
