### 关于`strlen()`函数的返回值  
库函数`strlen()`的返回值为`size_t`类型（即`unsigned int`），其返回类型计算时结果仍为`size_t`类型  

如:
```C
int main()
{
	if (strlen("abc") - strlen("abcd") > 0)
	{
		printf("hehe\n");
	}
	else
	{
		printf("haha\n");
	}

	return 0;
}
```
程序最终输出结果为**hehe**

### 关于`=`与`==`的陷阱！！！
`=`为运算操作符，意为将右值赋给左边（如 `a = 0`）
`==`为判断操作符，意为判断两侧值是否相等（如 `a == 0`）
在例如循环的条件判断中，经常会用到`==`来判断条件是否成立，为误写为`=`,可以将常量放到`==`右侧（如 `0 == a`）来判断，因为`0 = a`是错误语法（变量不可能赋给常量），会报错，使得程序提前发现错误且便于调试
